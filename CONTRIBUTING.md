# Contributing to Ollama Paperclip

We welcome contributions! As this is a v1 stable release, we are looking for:
- Bug fixes related to task orchestration.
- New specialized agent prompts.
- UI/UX improvements (using Vanilla CSS).
- Support for additional local LLM providers.

## Development Setup
1. Clone the repo.
2. Run `npm install`.
3. Start the dev server with `npm run dev`.
4. Ensure your local Ollama instance is accessible.

## Pull Request Process
1. Create a branch for your feature.
2. Ensure `npx tsc --noEmit` passes without errors.
3. Submit a PR with a clear description of the changes.
